# Griff dos Guris

Landing page institucional desenvolvida para a loja **Griff dos Guris**, localizada em São Borja - RS.

O projeto foi criado com foco em apresentação da loja, divulgação de produtos e conversão de clientes através do WhatsApp e Instagram.

---

## Sobre o projeto

A Griff dos Guris é uma loja de roupas e acessórios masculinos voltada para o público que busca estilo, qualidade e atendimento personalizado.

Esta landing page possui como objetivo:

* Apresentar a loja.
* Exibir produtos em destaque.
* Mostrar os diferenciais da empresa.
* Gerar contatos pelo WhatsApp.
* Divulgar o Instagram da loja.

O projeto não possui carrinho de compras, login ou sistema de pagamentos.

## Como executar

1. Clone o repositório:

```bash

```

2. Acesse a pasta:


3. Abra o arquivo:


Ou utilize a extensão **Live Server** no Visual Studio Code.
